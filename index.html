How It Works

1

Pick a Game

Click any game above. It opens right in your browser — nothing to install.

2

Add to OBS

Add as a Browser Source with ?admin=false&bg=transparent for a clean overlay.

3

Add Names

Type viewer names in the admin panel, or connect a chat bot via the postMessage API.

4

Go Live

Hit Start and let the game run. Winners announced automatically.
